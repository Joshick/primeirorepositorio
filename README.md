# como criar um repositório no Github
é necessário ter instalado o **GIT** no computador
vamos usar também o **GitHub no Desktopo** para facilitar

## Passo a passo
Supondo que você já tem um projeto criado, faça o seguinte:
- abrir o cmd dentro da pasta  e digitar o comando `` git init``
- Ir no GitHub Desktop e escplher a opção `` add existing repository`` e selecionar a pasta do projeto
- por fim fazer o commit e publicar. **atenção** lembrar de deixar sempre público
